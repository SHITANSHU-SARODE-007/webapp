Node Samples

NodeJS Express Web Application for the list of laptops.

Previous Next

Node samples Application Node App. This application uses node_modules to perform required NodeJS request made by th eRestful web services Application is been deployed on local host as http://localhost:3000/ & in NodeJs command prompt I fired the application by using node command as npm start as you can see in the slider above in images. All HTML tags and page structure is HTML5 compliant • I Used the express-generator node package (and express command) to generate my project. • Then I Created a new route file that holds the routes for the application. • Data for the application is selected such that each object should have at least 4 properties plus id. For example,A laptop may have id, color, name, price, title. Just hard coded the data in the routes file. The project have at least 4 entries as dummy data. • I defined two REST routes at a minimum. I chose laptops & they would probably be: /laptops to show list of all laptop objects. /laptops/1 to show the details of one specific id of a laptop. • Created the proper Jade templates to display both the list of all objects and the specific object display in a nicely styled and formatted way with css. • I used bootstrap css for the application. It is linked to the cdn or I can download the css files to the public stylesheets folder and link there. • Important part here is I link to the route for the object list to the main index homepage route so when we run the add and visit localhost:3000 we see the default index page with a button that takes us to the list route page. Also produced two additional routes in a separate route file for an api that returns JSON data. So one route that returns all the objects as a JSON array and one route that returns the individual JSON object that aligns with the id passed in the url.
